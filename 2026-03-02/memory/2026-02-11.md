## 学习记录：OpenClaw「永不失忆」记忆系统
- **来源**: https://x.com/calicastle/status/2021229394724102229 (Cali Castle)
- **标题**: 如何给 OpenClaw 搭建一套「永不失忆」的记忆系统
- **核心**: 从丢失两天记忆到三层自动化架构的踩坑教程
- **关键技术**: QMD Memory Backend (`memory.backend = "qmd"`)
  - 本地优先搜索 sidecar，结合 BM25 + 向量搜索 + 重排序
  - Markdown 仍是数据源，QMD 负责语义检索
  - 使用本地嵌入模型，不需要 OpenAI API
- **三层架构**: 短期记忆(daily md) + 长期记忆(MEMORY.md) + 语义检索(QMD)
- **启示**: 我们可以考虑启用 QMD 后端来增强记忆搜索能力
  - 当前 memory_search 因缺少 embedding API key 而不可用
  - QMD 使用本地模型，可以解决这个问题
