### AI 提示词整理
- 数据库里的 system_prompt 有严重的多层转义问题（`\\\\n` 混 `\n`）
- 原因：通过 SSH heredoc 写中文到 Python 再写 SQLite，每层都加转义
- **解决方案**：本地写 Python 文件 → scp 传到服务器 → python3 执行，避免 shell 转义
- 用 DELETE + INSERT 替代 UPDATE 确保干净写入
- 最终提示词格式：分块【身份】【最重要的规则】【什么时候回复】【上下文提示】【特殊能力：表情包】【特殊能力：语音】【消息格式】

### AI 回复控制 - [SKIP] 机制
- AI 之前每条消息都回复（代码没有跳过逻辑）
- 在 ai.js 的 reply 返回前加了 [SKIP] 检测：AI 回复 `[SKIP]` 则返回 null，不发消息
- processAIMessage 处理 null reply 返回 `{ok:true, skipped:true}`

### AI 语音主动发送
