# BabyNamePick: I Built a Free AI Baby Name Generator with 16 Cultural Origins

*As a solo developer, I built BabyNamePick.com — a free baby name generator that helps expecting parents find meaningful names from 16 different cultural traditions.*

## The Problem

When my friend was expecting, she spent weeks scrolling through generic baby name lists. The existing tools were either:
- Bloated with ads and pop-ups
- Limited to Western names only
- Behind paywalls for "premium" features
- Lacking cultural context and meanings

## What I Built

**BabyNamePick** (babynamepick.com) is a 100% free, no-signup baby name generator with:

- **16 cultural origin pages**: Japanese, Korean, Chinese, Indian, Arabic, Spanish, African, Celtic, Hawaiian, Greek, Biblical, and Royal names
- **Detailed meanings**: Every name comes with its meaning, origin story, and cultural significance
- **Smart filtering**: Filter by gender, starting letter, origin, and name style
- **SEO-optimized content**: Each cultural page has 500+ words of original content about naming traditions

## Tech Stack

- **Next.js 16** with static export (`output: "export"`)
- **Tailwind CSS v4**
- **Vercel** for hosting (free tier, global CDN)
- **No backend** — everything runs client-side

## The Journey

Built the entire site in 2 days:
- Day 1: Research, domain purchase (babynamepick.com on Namecheap), built core app, deployed to Vercel
- Day 2: Added 16 cultural sub-pages, fixed Tailwind v4 compatibility, set up Google Search Console
