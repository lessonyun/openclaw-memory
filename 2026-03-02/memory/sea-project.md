## 完整执行路线

### Phase 1：准备期（第1-2周）⬅️ 当前
**目标：把工具准备好，开始找词**

**待办清单：**
- [ ] 注册 Google Search Console
- [ ] 注册 Google Analytics
- [ ] 注册 Ahrefs Webmaster Tools（免费版）
- [ ] 安装 AITDK 浏览器插件
- [ ] 安装 Similarweb 浏览器插件
- [ ] 熟悉 Google Trends 操作
- [ ] 建立根词列表（Google Sheets 或 Notion）
- [ ] 确定大方向：AI工具 / 在线工具 / 游戏 / 其他

**每日任务（10-15分钟）：**
- 在 Google Trends 用根词查看相关推荐
- 记录有潜力的新词到根词列表
- 看1-2个竞品站，用AITDK查流量

**里程碑：** 积累 20+ 个候选关键词

---

### Phase 2：选词定方向（第3-4周）
**目标：选定1-2个关键词，注册域名**

**待办清单：**
- [ ] 对候选关键词逐一用5步判断法评估
- [ ] 筛选出3-5个最有潜力的词
- [ ] 深入分析竞品（流量、外链、页面结构）
- [ ] 最终选定1-2个关键词
- [ ] 注册 .com 域名（关键词相关）
- [ ] 确定技术方案（Next.js + Vercel/Cloudflare）

**关键决策点：**
- 关键词月搜索量 > 1000
- KD 不要太高（新手建议 < 30）
- 搜索结果前10有内页（有机会）
- 不是品牌词/公司名

**里程碑：** 域名注册完成，方向确定

---

### Phase 3：建站（第5-8周）
**目标：做出第一个精品工具页面**

**待办清单：**
- [ ] 搭建 Next.js 项目框架
- [ ] 开发核心工具功能
- [ ] 按 V2.0 模式设计精品工具页面（工具+落地页+结果展示三合一）
- [ ] 写高质量落地页文案（围绕关键词）
- [ ] On-Page SEO 优化（TDK、H标签、内链、Schema）
- [ ] 移动端适配
- [ ] 页面速度优化
- [ ] 部署上线（Vercel / Cloudflare Pages）

**技术要点：**
- 必须 SSR（后端渲染），不能纯 CSR
- 一个关键词一个页面
- 前后台统一设计
- 结构化数据 Schema 标记

**里程碑：** 网站上线，能正常访问

---

### Phase 4：推广收录（第9-20周）
**目标：被Google收录，开始拿排名**

**上线后第一周：**
- [ ] 提交 Google Search Console + sitemap
- [ ] 每天发 3-5 个博客评论外链
- [ ] 监控收录情况（24-48小时应该被收录）

**持续执行（每周）：**
- [ ] 每周新增 10-20 个外链（博客评论为主）
- [ ] 每2周新增 1-2 个内页（覆盖长尾词）
- [ ] 每周检查 GSC 出词情况
- [ ] 根据数据调整关键词和内容
- [ ] 抄竞品外链作业（Ahrefs导出竞品外链）

**关键指标监控：**
- GSC 收录页面数
- GSC 平均排名
- 关键词排名变化
- 网站流量（GA）

**里程碑：** 核心关键词进入 Google 前50

---
