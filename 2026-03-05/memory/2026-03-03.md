- **原因**：只改了全局 `~/.openclaw/openclaw.json` 的 anthropic baseUrl/apiKey
- **遗漏**：agent 级别配置优先级更高，未同步更新：
  - `~/.openclaw/agents/main/agent/models.json` — baseUrl 仍指向旧中转
  - `~/.openclaw/agents/main/agent/auth-profiles.json` — auth 凭据仍是旧的 cr_ token
- **教训**：切换中转站必须同时更新全局配置 + agent 级别配置，缺一不可
- **正确操作顺序**：
  1. `openclaw gateway stop`
  2. 改 `openclaw.json`（全局）
  3. 改 `agents/main/agent/models.json`（agent 级别）
  4. 改 `agents/main/agent/auth-profiles.json`（auth profile）
  5. 重启 gateway
