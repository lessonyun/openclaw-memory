### 技术问题及解决

**1. Discord MESSAGE CONTENT INTENT**
- 五一和longxia都遇到收不到图片/消息内容的问题
- 原因: Discord Developer Portal需要开启MESSAGE CONTENT INTENT
- 位置: Bot → Privileged Gateway Intents → MESSAGE CONTENT INTENT

**2. longxia图片识别失败**
- 配置了`input: ["text", "image"]`但还是看不到图片
- 根本原因: 第三方API(ltcraft - ai.ltcraft.cn)不支持图片content block
- 需要换支持视觉的API端点

**3. 五一私聊问题**
- dm.policy设为"open"时，allowFrom必须包含"*"
- 解决: 改回`"policy": "pairing"`
- 用Python脚本修改配置文件

**4. Chrome浏览器控制**
- 桔梗的Chrome在外置硬盘: `/Volumes/外置2t/Applications/Google Chrome.app`
- 启动带调试端口: `open "/path/to/Chrome.app" --args --remote-debugging-port=9222`
