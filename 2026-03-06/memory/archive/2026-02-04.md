## 经验教训
- **子任务会抢浏览器标签**: 子任务和主任务共享同一个浏览器 profile，会互相导航对方的标签页。以后不要同时在浏览器上跑多个任务
- **Ant Design 文件上传**: browser tool 的 upload 动作对 display:none 的 file input 不触发 React change event，必须用 CDP websocket 的 `DOM.setFileInputFiles` 才行
- **推客密码页面不稳定**: 同时开抖音等重页面会导致标签崩溃，操作推客密码时关掉其他标签
- **协议复选框**: tuikemima 登录页的复选框需要用 `input[type=checkbox]` 直接 click，不能点外层容器
- **Rate limit**: Opus 4.5 在密集操作后容易触发 429，需要等待恢复
- **screencapture 黑屏**: launchd 服务无屏幕录制权限，创建 App 包装器 `/Users/yunhan/Applications/OpenClaw Screen.app` 解决
- **服务器 Docker**: OpenCloudOS 不支持官方 docker 安装脚本，用 `yum install -y docker` 安装
- **SSH expect**: 用 expect 脚本处理 ssh 密码输入，注意特殊字符转义（如 `\]`）
- **better-sqlite3 在 Docker 中编译失败**: OpenCloudOS 的 node:20-slim 镜像缺少编译工具，换用纯 JS 的 sql.js 替代
- **云寒强调**: 做一步汇报一步，不要卡住不说话！需要权限就自己点或提醒他
