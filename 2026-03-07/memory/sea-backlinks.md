# 外链建设策略 - babynamepick.com

## 目标
- 55个referring domains（超过 baiby.co 当前水平）
- 3-5个外链/天，持续2个月

## 外链渠道

### 1. 育儿论坛（高相关性）
- **BabyCenter Community** (community.babycenter.com) - Baby Names 板块
- **What To Expect** (community.whattoexpect.com) - Baby Names 讨论区
- **The Bump** (forums.thebump.com) - Baby Names 板块
- **BabyAndBump** (babyandbump.momtastic.com) - Baby Names 板块
- **Reddit** r/namenerds, r/BabyNames, r/pregnant

### 2. 问答平台
- **Quora** - "What are good baby names" 类问题
- **Reddit** - r/namenerds 是最活跃的起名社区

### 3. 目录提交
- Product Hunt (提交为 AI tool)
- AlternativeTo (作为 baby name generator 替代品)
- ToolsForHumans, FutureTools, AIToolsDirectory
- StartupBase, BetaList

### 4. 博客评论（nofollow 但增加曝光）
- 育儿博客文章底部留言
- 搜索 "best baby names 2026" 找到相关博客

### 5. Guest Post 机会
- 联系育儿博客写客座文章
- 提供 "How to Choose a Baby Name" 类内容

### 6. Social Bookmarking
- Pinterest（创建 baby names 图片 pin）
- Twitter/X 分享起名建议

## 执行节奏
- 第1-2周：论坛注册 + 目录提交（10-15个）
- 第3-4周：Reddit/Quora 回答问题（每天2-3个）
- 第5-8周：Guest post 联系 + Pinterest 内容

### 7. AI 工具目录提交（高优先级，快速获取 dofollow 外链）
- **Product Hunt** (producthunt.com/launch) - 直接提交
- **FutureTools** (futuretools.io) - 免费提交
- **There's An AI For That** (theresanaiforthat.com) - 提交AI工具
- **AI Tool Directory** (github.com/best-of-ai/ai-directories) - 93+ 目录列表
- **ENUM** (enumhq.com/directory-list) - 目录集合
- **The AI Surf** (theaisurf.com) - 提交AI公司
- **FreeAITool** (freeaitool.ai/submit) - 免费AI工具
- **Futurepedia** (futurepedia.io) - AI工具百科
- **ToolPilot** (toolpilot.ai) - AI工具导航

## 注意事项
- 不要硬推链接，先提供价值再自然提及工具
- Reddit 尤其反感 spam，要先建立 karma
- 论坛先贡献几个有价值的回答再提及网站
