## 第一阶段：Ahrefs SEO基础（7篇，已学完）

### 1. 搜索引擎如何工作
- 搜索引擎 = 搜索索引 + 搜索算法
- Google流程：URL发现 → 抓取 → 处理&渲染 → 索引
- URL来源：外链、站点地图、URL提交
- 6大排名因素：外链、相关性、新鲜度、话题权威性、页面速度、移动友好
- 外链是最重要排名因素之一，权重+相关性是关键
- 搜索结果会根据位置、语言、搜索历史个性化

### 2. SEO基础
- SEO重要因为自然搜索流量 > 所有其他流量总和
- SEO三大好处：更多流量、稳定流量、"免费"流量
- 网站基础设置：好域名(.com优先)、WordPress推荐、好主机、HTTPS、移动友好
- 站点逻辑结构 + 内部链接很重要
- URL结构推荐用文章名称格式
- SEO四大方面：关键词研究、页面SEO、链接建设、技术SEO
- 跟踪指标：自然流量、声音份额(SoV)、转化率

### 3. 关键词研究
- 种子关键词 → 关键词工具扩展 → 竞争对手分析
- 五大关键词指标：搜索量、点击、流量潜力、关键词难度(KD)、CPC
- 搜索量是年度月平均，要看趋势图
- 流量潜力要看排名第一页面的实际流量，而非单个词搜索量
- KD基于外链数量，但不能只做低KD的词，高KD词要尽早布局
- CPC反映商业价值
- 父话题分组：检查Google是否把多个关键词视为同一话题
- 搜索意图3C：Content type/format/angle
- 商业潜力评估：TOFU/MOFU/BOFU漏斗

### 4. SEO Content（英文）
- 选择有流量潜力、商业潜力、排名潜力的话题
- 分析搜索意图的3C
- EEAT：经验、专业、权威、信任
- 全面覆盖话题（exhaustive），但不等于长
- 独特性：原创研究、专家观点、实验、数据
- 粘性：简单语言、视觉化、校对
- 保持内容常青，定期更新

### 5. 页面SEO（On-Page SEO）
- 与搜索意图对齐是核心
- 全面覆盖子话题（Content Gap分析）
- 内容独特性：原创研究、独特观点、免费资源
- H1-H6标签层次结构
- 标题标签：≤70字符，描述性，含关键词
- 元描述：≤160字符，扩展标题，主动语态
- URL slug用关键词
- 图片优化：描述性文件名、alt文本、压缩
- 内部链接建设
- 外部链接：引用来源，nofollow付费链接
- 特色片段优化
- Schema标记获取富媒体片段
- 页面体验：CWV、HTTPS、移动友好

### 6. 链接建设
- 链接是Google三大排名因素之一
- 四种方式：添加链接、请求链接、购买链接(不推荐)、赚取链接
- 添加链接：目录、社交资料、博客评论、论坛（价值低但有用）
- 请求链接：客座博客、摩天楼技术、链接植入、失效链接建设等
- 赚取链接：创建出色内容 + 推广
- 维护链接：修复404、监测丢失链接
- 链接评估：权威性、相关性、锚文本、follow/nofollow、位置、目标
- 四大推荐策略：竞争对手链接分析、可链接资产、内容推广、客座博客

### 7. 技术SEO
- 确保搜索引擎能查找、抓取、理解、索引页面
- 抓取控制：robots.txt、抓取频率
- 索引：爬虫指令(noindex等)、规范化(canonical)
- 优先事项：检查索引、回收丢失链接(301重定向)、内链、Schema标记
- 页面体验：CWV、HTTPS、移动友好、安全浏览
- Hreflang用于多语言
- 维护：修复失效链接、重定向链

## 重点文章详细笔记（登录后阅读）

### 小白怎么找词（睡觉想飞）
- 基于哥飞找词8种方式的视频教程
- **谷歌趋势找词法**：用根词（如AI）→ 查看相关推荐/相关主题 → 点开可能做成工具的词 → 关注搜索上升量
- **新词扩展**：新词当根词 → 调小时间看相关推荐 → 扩展出新词或新根词
- **根词维护**：保存根词列表 → 7天/30天周期查询 → 能扩展就保留，不能就剔除
- **竞争判断**：公众号公开的词、域名被大量注册的词，不建议做

### 如何判断关键词是否能做（哥飞）
**核心5步判断法：**
1. 看搜索量（谷歌趋势、Google Ads关键词规划工具、Similarweb）
2. 看搜索结果（前10是首页还是内页，内页=机会大）
3. 看网站流量（AITDK插件、Similarweb、Ahrefs）
4. 看网页流量（Similarweb看具体页面）
5. 看外链数量（Ahrefs Backlink查询）

**关键判断原则：**
- 搜索量小+竞争大 → 完全不考虑
- 搜索量大+竞争大 → 可以做（蛋糕够大，分一小块也够）
- 公司/品牌名 → 不做（如stargate ai，做了=钓鱼网站）
- 投资回报率：流量÷外链数量，太低就不值得单独做站
- **不要被高KD吓倒**，AI新词即使KD六七十也可以做

### 上站之后要做什么（哥飞）⭐超重要
**前提：已做好关键词研究+围绕关键词建站**

**上站流程：**
1. 提交GSC → 等待收录（24小时内）
2. 加外链加速收录（最快2小时）
3. 收录后24-48小时内看GSC出词
4. 没出词？排查：内容长度/关键词分布/TDH/前端渲染问题
5. 出词后看是否是目标关键词
6. 持续加外链提升排名

**外链策略：**
- 新站：先量后质，有外链就是零的突破
- 方法：搜竞品 → Ahrefs/Semrush导出竞品外链 → 取并集 → 越多站都有的外链越优先做
- 权重越高后，需要的外链质量越高

**上站后核心两件事：**
1. 加精品页面（不是批量垃圾页面）
2. 加外链（越多越好，不买垃圾外链）

**关键用户行为指标：**
- 跳出率→要低
- 停留时间→要高
- 人均访问页面次数→要多

**新词优势：** 参与排名的200个网页大多是弱鸡，先发优势巨大

**Sprunkin.com案例（月入$50k）：**
- 后来居上，域名晚买20天但坚决入场
- 三板斧：不断加外链(878个) + 不断加内页(1330页) + CDN托管游戏iframe引流
- 3个月从3万涨到365万月访问量

### 精品工具页面思路（哥飞）⭐核心方法论
**三个公式：**
1. 工具页 + 落地页 + 结果展示页 = 精品工具页面
2. 精品工具页面 + 低竞争度关键词 = 排名和曝光
3. 排名和曝光 + 优秀用户行为数据 = 搜索结果前几名

**前提：一个关键词 = 一个页面**（同一工具可做多个页面，对应不同关键词说法）

**进阶五公式：**
- 一个关键词 = 一个页面
- 工具页+落地页+结果展示页 = 精品工具页面
- 精品工具页面+低竞争度关键词 = 排名和曝光
- 工具页面前后台统一+用户行为数据 = 在一个页面积累足够多优秀用户行为数据
- 排名和曝光+优秀用户行为数据 = 搜索结果前几名

**John的流量排名公式：SEO = 内容 × 外链 × 用户体验**
- 进入前10靠外链，前10排名靠体验

**海量页面策略（白名单标签聚合模式）：**
- UGC页面设noindex → 不参与排名
- 为有搜索量的关键词做标签聚合页 → 文案+图片列表+工具入口
- ⚠️ 新站禁止搞海量页面（必死）
- 要求：网站上线半年以上+有权重+有外链+慢慢上

**前后台统一：** 同一URL，登录前显示落地页图文，登录后显示工具界面（参考pollo.ai）

### 博客评论外链（哥飞）
- 在WP博客评论区留链接，个人主页和姓名里包含链接即可
- 最好留有价值的讨论评论，提高站长审核通过率
- **抄作业法**：看竞品外链列表 → 找到他们发过评论的博客 → 你也去发
- nofollow外链也比没有好
- 可以半自动化/自动化批量发

### 博客评论外链衍生的需求挖掘新思路
- 所有去发外链的网站都是有利可图的
- 抓取评论区里的网站 → 查域名注册时间+流量 → 发现别人偷偷赚钱的产品
- 分析产品和关键词 → 获得灵感

### 落地页V1.0到V2.0演进（哥飞）
- **V1.0**：工具页面 + 博客页面（早期万兴模式）
- **V1.5**：工具页面 + 落地页 + 博客页面（为每个关键词做专门落地页）
- **V2.0**：精品工具页面 = 工具功能 + 落地页图文 + 工具生成结果列表

**V2.0核心思想：**
- 工具、落地页、博客都在一个域名下（/blog子目录）
- 落地页和工具页合二为一
- 用户在一个页面完成所有操作
- 展示精选UGC结果（算法筛选，非人工）

**V2.0优势：**
1. 满足谷歌用户体验要求
2. 停留时间长 → 提升排名
3. 减少跳转 → 提升转化率
4. 动态内容更新 → 页面常新

**注意事项：**
- UGC内容不要直接生成海量页面（浪费爬虫预算→被惩罚）
- 用/tag/标签页做精品聚合，而非每个UGC结果一个页面
- Prompt太长要用AI总结，避免稀释关键词密度
- 导航栏引导用户访问其他页面（降低跳出率）

## 核心要点总结

### SEO本质
1. **搜索引擎的目的是给用户最好的结果** → 你的目标是成为那个最好的结果
2. **内容 + 链接 = 排名的核心** → Google高管亲口确认
3. **搜索意图是一切的起点** → 不匹配意图，其他全白搭

### 出海工具站方法论（哥飞体系）
1. **找词** → 用Ahrefs/Google趋势/竞品分析挖掘有搜索量的需求
2. **判断** → 评估关键词难度、商业价值、是否可做
3. **建站** → 用AI(ChatGPT/Cursor)快速生成网页
4. **优化** → On-Page SEO + 内链 + 多语言
5. **部署** → Vercel/Cloudflare + 域名
6. **推广** → GSC提交 + 外链建设(博客评论等) + 内容营销
7. **变现** → 广告(Adsense) / SaaS订阅(Stripe) / 其他

### 关键经验
- 新手先从低KD词入手，但不要放弃高KD词的布局
- 多上站，多实践，不要只看不做
- AI生成内容可以用，但要确保质量
- 外链很重要，博客评论外链是低成本起步方式
- 养网站是长期事业，要有耐心
